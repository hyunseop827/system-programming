# `fopen()` mode
fopen is high-level File IO.

|mode|fucntion|
|----|--------|
|`r`|open file as read only|
|`w`|open text file as write, original contents will be deleted|
|`a`|open text file as append mode|
|`rb`|open binary file as read only|
|`wb`|open binary file as write, original contents will be deleted|
|`ab`|open binary file as append mode|
|`r+`|open text file as read/write|
|`w+`|open text file as write/read|
|`a+'|open text file as append/read|
|`rb+`|open binary file as read/write|
|`wb+`|open binary file as write/read|
|`ab+`|open binary file as append/read|
